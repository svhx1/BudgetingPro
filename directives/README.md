# Directive Layer (Layer 1)

Esta pasta contém os POPs (Procedimentos Operacionais Padrão) em formato Markdown.

## O Que Fazer Aqui
- Defina os objetivos, entradas, ferramentas/scripts a usar, saídas e casos extremos (edge cases).
- Escreva instruções em linguagem natural, como se estivesse orientando um funcionário de nível pleno.

**Nota:** O agente Orquestrador (Layer 2) lerá estas diretrizes para tomar decisões de roteamento inteligentes e acionar as ferramentas de execução corretas em `execution/`.
